From XOWA: the free, open-source, offline wiki application
 

Font

 
Enabled
 

Choose one of the following

  • checked : Enables custom HTML page font
  • unchecked : Disables custom HTML page font. The HTML page font will be the browser default (Arial 16)
 
Name
 

Enter a font family name.

 
Size
 

Enter a number representing for a valid font size in pixels.

 
Format
 

Custom

 
CSS
 

Enter a format for embedding the custom font info in the web page.
