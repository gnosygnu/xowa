Hardware

  • CPU: 1.0 GHz minimum; 2.0 GHz recommended
  • Memory: 512 MB minimum; 2 GB recommended
  • Storage: varies per wiki. As a rough guideline, multiply the dump file by 2. For dump sizes, see either Dashboard/Import/Online or http://dumps.wikimedia.org/backup-index.html.
    • For example, for English Wikipedia, the dump size is 12 GB.
      • The XOWA wiki will be about 25 GB
      • The import process will need about 2 GB in temp files.
      • You should have at least 30 GB available for the text only version. Images will require at least another 80 GB.

Software

  • Operating System
    • Windows 32 bit / 64 bit
    • Linux 32 bit / 64 bit
    • Mac OS X 32 bit / 64 bit
    • Raspberry PI (ARM)
    • Android
  • Java 1.7+
    • If Java is not installed on your machine, you can get it from http://www.java.com/en/download/manual.jsp
    • If you don't want to install Java, you can copy it from an existing machine, or download a portable package. See App/Setup/Flash card
    • On Debian Linux systems you can do sudo apt-get install openjdk-7-jre
