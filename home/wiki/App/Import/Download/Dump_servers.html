Dump servers

The list of dump servers is from http://meta.wikimedia.org/wiki/Mirroring_Wikimedia_project_XML_dumps

  • http://dumps.wikimedia.org/: The official dump server for Wikimedia.
  • http://dumps.wikimedia.your.org/: An alternate dump server. It is a complete mirror of the official dump server
  • http://wikipedia.c3sl.ufpr.br/: An alternate dump server (located in Brazil). It has the most recent 5 dumps.
  • http://ftp.fi.muni.cz/pub/wikimedia/: An alternate dump server (located in the Czech Republic). Its latest dump is as of 2012-11.

your.org

your.org is the default dump server. This mirror happens to be significantly faster than the official Wikimedia dump server. There was also a recent post on xmldatadumps-l that requested mirror server usage over the official ones: http://lists.wikimedia.org/pipermail/xmldatadumps-l/2012-December/000644.html
